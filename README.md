# 🎨 Etch A Sketch Pro+

A browser-based drawing app built using HTML, CSS, and JavaScript as part of The Odin Project Foundations curriculum.

## 🚀 Live Demo
https://sklogic-build.github.io/etch-a-sketch/

---

## 📌 Features

- 🖱️ Click and drag drawing
- 🎨 Pencil mode (default black drawing)
- 🌈 Rainbow mode (random colors)
- 🧽 Eraser tool
- 📏 Dynamic grid resizing (up to 100x100)
- 🧼 Clear grid button
- ⚡ Smooth and responsive UI

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox)
- JavaScript (DOM Manipulation & Events)
